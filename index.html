My work

03

Bachelor Project - Phone Application

AroundU is a solution I developed in my Bachelor team, it is a mobile application with the main feature to bring a virtual tour or houses and properties to the user. It is built on a N-tier architecture and it incorporates a lot of technologies. The presentation tier is build with Unity3D and it uses ARCore to take control of camera features. The logic tier that holds all the logic is a NodeJS api that incorporates libraries like ExpressJS, Mongoose ODM and many more to provide all the functionalities needed to manage 360 high quality pictures and other type of data. The storage tier is separated on multiple cloud databases MongoAtlas and Cloudinary. These cloud operating system provide storage for data and high quality images. Aditionally, other cloud services and APIs were used like google cloud functions and geocoder to improve the quality of the system.

Virtual Reality Games

During my last semester I have developed 2 Virtual Reality Games using Unity3D and a pair of Oculus Rift Glasses. One of them is called VirtualDJ, a game where you play the role of a DJ in a Sci-Fi club in space, you pick what songs to play, play with the light effects and access some cool fireworks. The other game is an escape room with the theme being the Chernobyl inspired by the HBO series. In the game you have to solve a puzzle to get access to couple of fuses that can prevent the reactor from exploding. You solve it in time, you save the reactor, you run out of time, you are responsible for the explosion. The game creates preassure on the player with sounds, timer, lightning and other styling elements.

FPS Shooter - A Russian Dream

First person shooter where the character is a russian man dreaming about shooting targets in a sci-fi enviroment. The weapon of chioce is gonna be an AK-47 more specifically a 3D free model with animations from the internet. This project is a combination of free assets from the internet and created assets by me. To build this game I had a journey trough the entire creative pipeline exploring scripting, sound, animation, visual effects, drawing textures, creating 3D models and many more creative activities. One of the game mechanics I implemented is wall-running which probably wasn't the best feature I could come up with but it was a fun experience either way.
